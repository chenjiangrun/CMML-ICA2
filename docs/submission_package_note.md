# Submission Package Note

Two submission packages are available.

## Recommended Upload

`miniproject7_3127_light_submission.zip`

This package is about 13 MB and contains:

- report DOCX and PDF
- all scripts
- all result tables
- all figures
- documentation and reproducibility notes

It does not include the processed count matrices, which keeps the upload small.
The real-data workflow and dataset conversion scripts are still included.

## Full Archive

`miniproject7_3127_submission.zip`

This package is about 260 MB and additionally contains:

- processed simulated datasets
- processed real datasets

It intentionally excludes the downloaded raw `real_datasets.zip` archive and
the extracted raw `.rds` files, because those files are large and can be
re-downloaded from Zenodo record 4562782.

Use this only if the submission system allows large uploads or if the marker
explicitly asks for processed matrices.
